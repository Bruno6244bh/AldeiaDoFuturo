Aula 1 - Introdução

Nesta aula vamos introduzir conceitos básicos para a criação de jogos, além de um breve resumo sobre os tópicos iniciais do curso

O que é um jogo ?

Um jogo é uma atividade estruturada, geralmente para entretenimento, mas também pode ser uma ferramenta de educação ou treinamento. Um jogo normalmente apresenta os seguintes elementos básicos:

  • Regras: Estruturas que definem o que é permitido ou não no jogo.

  • Objetivo: A meta que os jogadores devem alcançar.

  • Interatividade: A capacidade dos jogadores interagirem com o jogo e entre si.

  • Feedback: Respostas do jogo às ações dos jogadores (pontuações, progressos, etc.).

Unreal Engine

Unreal Engine é uma das ferramentas mais poderosas atualmente para a criação de jogos. Com ele, é possível criar os mais variados tipos de projetos, para diversas plataformas (Como PC, Consoles e Dispositivos Móveis) desenvolvendo gráficos ultrarealistas de forma prática. Conhecida pela facilidade de uso e aprendizado para novos usuários, está é a engine que trabalharemos durante boa parte do curso. Porém ela não é a unica, Hoje temos dezenas de engines capazes de desenvolver jogos de alta qualidade, nas quais todos os anos grandes titulos são lançados. Algumas delas são: Unity Engine, GameMaker Studio, RPG Maker, entre outras.

Grandes Títulos criados na Unreal Engine

Valorant, Fortnite, Ark, Gears of War, entre tantos outros títulos demonstram o poder de criar jogos incríves com o Unreal Engine

Blueprint

O Blueprint representa mais uma das praticidades ao ter o Unreal como primeira engine para criação de jogos. Ao invés da programação de jogos unicamente por linhas de código, o unreal fornece esta opção, que é uma modalidade de programação em blocos, mais visual e prática e que facilita o entendimento para programadores iniciantes
